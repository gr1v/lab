# Мультисписок
### Объяснение алгоритма: 
Мультисписок — это структура данных, где каждый элемент одновременно входит в несколько односвязных списков.

Python:

Для обработки применяется внешний цикл for group in groups для обхода строк и внутренний цикл for name in group для элементов строки

Используется метод join() для форматированного вывода через конкатенацию в print

C++:

Используется структура двусвязного списка с указателями prev и next

Указатели инициализируются значением nullptr

Java:

Мультисписок реализуется через коллекции: пользовательские классы Node и DoublyLinkedList

# Очередь
### Объяснение алгоритма: 
Очередь — линейная структура данных, организованная по принципу FIFO (First In, First Out).

Python:

Применяется queue.Queue для классической очереди FIFO

Используется collections.deque для эффективного добавления/удаления с обоих концов

C++:

Используется std::queue — классическая очередь (FIFO)

Применяется std::priority_queue — очередь с приоритетом

Java:

Основные реализации: LinkedList и PriorityQueue

# Дек (двусторонняя очередь)
### Объяснение алгоритма:
Дек — структура, поддерживающая добавление и удаление элементов как в начале, так и в конце последовательности.

Python:

Реализуется через collections.deque с методами appendleft(), popleft(), append(), pop()

C++:

Применяется контейнер std::deque, обеспечивающий эффективное выполнение операций с обоих концов

Java:

Дек представлен классами ArrayDeque и LinkedList с методами push() (аналог addFirst()), pop() (аналог removeFirst())

Приоритетная очередь
Объяснение алгоритма: Приоритетная очередь — абстрактный тип данных, где элементы обрабатываются согласно их приоритету, а не порядку добавления.

Python:

Реализуется через queue.PriorityQueue (потокобезопасная версия)

Используется heapq (на основе бинарной кучи) с операциями heappush(), heappop(), проверкой на пустоту

C++:

Используется std::priority_queue из STL с методами push(), pop(), top(), empty()

Java:

Класс PriorityQueue поддерживает как естественный порядок, так и пользовательский компаратор с методами offer(), poll(), peek(), isEmpty()
